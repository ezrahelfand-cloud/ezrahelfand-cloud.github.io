# Ezra Helfand — Minimalist Blue Portfolio

A rubric-perfect, minimalist portfolio for the Catalyst Program built with **React + Vite + Tailwind**.

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

The output will be in `dist/` and can be deployed to Netlify, Vercel, or GitHub Pages.

## Deploy — quick pointers
- **Netlify**: Build `npm run build`, publish directory `dist/`
- **Vercel**: Import repo, build `npm run build`, output `dist`
- **GitHub Pages**: Use `gh-pages` and set `"deploy": "gh-pages -d dist"` in `package.json`
